
# ERM-System Documentation

рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃљрЃарЃБрЃџрЃў **ERM рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃДрЃЋрЃћрЃџрЃљ рЃўрЃЏ рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃА, рЃарЃљрЃф рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЉрЃўрЃќрЃюрЃћрЃАрЃА рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃћрЃарЃЌрЃўрЃљрЃю рЃћрЃЎрЃЮрЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў:
- **CRM** РђЊ рЃЎрЃџрЃўрЃћрЃюрЃбрЃћрЃЉрЃЌрЃљрЃю рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЮрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ рЃЊрЃљ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃљ
- **рЃЉрЃБрЃдрЃљрЃџрЃбрЃћрЃарЃўрЃљ** РђЊ рЃцрЃўрЃюрЃљрЃюрЃАрЃБрЃарЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃљрЃдрЃарЃўрЃфрЃ«рЃЋрЃљ рЃЊрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃљрЃюрЃњрЃљрЃарЃўрЃерЃњрЃћрЃЉрЃљ
- **рЃАрЃљрЃгрЃДрЃЮрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ** РђЊ рЃЏрЃљрЃарЃљрЃњрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў рЃЊрЃљ рЃџрЃЮрЃњрЃўрЃАрЃбрЃўрЃЎрЃБрЃарЃў рЃърЃарЃЮрЃфрЃћрЃАрЃћрЃЉрЃўрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃљ
- **рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ** РђЊ рЃърЃарЃЮрЃЊрЃБрЃЦрЃфрЃўрЃўрЃАрЃљ рЃЊрЃљ рЃюрЃћрЃЊрЃџрЃћрЃБрЃџрЃўрЃА рЃЎрЃљрЃбрЃљрЃџрЃЮрЃњрЃўрЃќрЃљрЃфрЃўрЃљ, рЃцрЃљрЃАрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃерЃћрЃЎрЃЋрЃћрЃЌрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃљ
- **рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃћрЃЉрЃў рЃЉрЃљрЃюрЃЎрЃћрЃЉрЃЌрЃљрЃю рЃЊрЃљ рЃАрЃљрЃ«рЃћрЃџрЃЏрЃгрЃўрЃцрЃЮ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃЌрЃљрЃю** РђЊ RS.ge, рЃАрЃљрЃњрЃљрЃЊрЃљрЃАрЃљрЃ«рЃљрЃЊрЃЮ рЃљрЃюрЃњрЃљрЃарЃўрЃерЃњрЃћрЃЉрЃљ рЃЊрЃљ рЃАрЃљрЃЉрЃљрЃюрЃЎрЃЮ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃўрЃќрЃљрЃфрЃўрЃљ рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў
- рЃЊрЃљ рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃАрЃ«рЃЋрЃљ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃерЃћрЃљрЃарЃЕрЃўрЃЮрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЉрЃўрЃќрЃюрЃћрЃАрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ.
рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃА рЃћрЃарЃЌ рЃАрЃўрЃЋрЃарЃфрЃћрЃерЃў, рЃ«рЃЊрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃћрЃЉрЃА рЃњрЃљрЃЏрЃГрЃЋрЃўрЃарЃЋрЃљрЃџрЃћрЃАрЃљ рЃЊрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЋрЃА, рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА рЃЊрЃарЃЮрЃўрЃАрЃљ рЃЊрЃљ рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃ«рЃљрЃарЃ»рЃА рЃЊрЃљ рЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљрЃЌ рЃЏрЃўрЃўрЃдрЃЮрЃЌ рЃБрЃцрЃарЃЮ рЃАрЃгрЃарЃљрЃцрЃў рЃЊрЃљ рЃАрЃгрЃЮрЃарЃў рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў.
­ЪЉЅ рЃўрЃ«рЃўрЃџрЃћрЃЌ рЃАрЃарЃБрЃџрЃљрЃЊ: [рЃЎрЃЋрЃџрЃћрЃЋрЃћрЃЉрЃў рЃЊрЃљ рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљ](docs/about/research.md)

## рЃЕрЃЋрЃћрЃюрЃў рЃЏрЃўрЃАрЃўрЃљ
рЃЕрЃЋрЃћрЃюрЃў рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ, рЃЊрЃљрЃЋрЃћрЃ«рЃЏрЃљрЃарЃЮрЃЌ рЃЦрЃљрЃарЃЌрЃБрЃџ рЃЏрЃфрЃўрЃарЃћ рЃЊрЃљ рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃЉрЃўрЃќрЃюрЃћрЃАрЃћрЃЉрЃА рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃфрЃўрЃцрЃарЃБрЃџрЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃЌ, рЃарЃљрЃЌрЃљ рЃњрЃљрЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃЮрЃю рЃДрЃЮрЃЋрЃћрЃџрЃЊрЃдрЃўрЃБрЃарЃў рЃАрЃљрЃЦрЃЏрЃўрЃљрЃюрЃЮрЃЉрЃљ, рЃерЃћрЃљрЃЏрЃфрЃўрЃарЃЮрЃю рЃЉрЃўрЃБрЃарЃЮрЃЎрЃарЃљрЃбрЃўрЃљрЃќрЃћ рЃЊрЃљрЃ«рЃљрЃарЃ»рЃБрЃџрЃў рЃЊрЃарЃЮ рЃЊрЃљ рЃерЃћрЃЊрЃћрЃњрЃљрЃЊ рЃњрЃљрЃќрЃљрЃарЃЊрЃЮрЃю рЃћрЃцрЃћрЃЦрЃбрЃўрЃљрЃюрЃЮрЃЉрЃљ.

## рЃарЃљ рЃерЃћрЃЊрЃћрЃњрЃА рЃЏрЃўрЃЋрЃљрЃдрЃгрЃћрЃЋрЃЌ
- рЃЉрЃўрЃќрЃюрЃћрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃљ  
- рЃЊрЃарЃЮрЃўрЃАрЃљ рЃЊрЃљ рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃЊрЃљрЃќрЃЮрЃњрЃЋрЃљ  
- рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃћрЃарЃЌрЃўрЃљрЃюрЃљрЃЊ рЃЊрЃљ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃЊ рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ  
- рЃАрЃгрЃарЃљрЃцрЃў рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃљ рЃЊрЃљ рЃАрЃгрЃЮрЃарЃў рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃџрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ

## рЃарЃўрЃЌ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃЊрЃћрЃЉрЃљ рЃЕрЃЋрЃћрЃюрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃЎрЃЮрЃюрЃЎрЃБрЃарЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃњрЃљрЃю
- **рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃЉрЃљрЃќрЃарЃўрЃАрЃЌрЃЋрЃўрЃА рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃў**  
- рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў рЃЦрЃљрЃарЃЌрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ  
- рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ Рђћ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃљрЃарЃЕрЃћрЃЋрЃљ  
- рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ рЃЉрЃљрЃюрЃЎрЃћрЃЉрЃЌрЃљрЃю рЃЊрЃљ рЃАрЃљрЃ«рЃћрЃџрЃЏрЃгрЃўрЃцрЃЮ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃЌрЃљрЃю (RS.ge рЃЊрЃљ рЃАрЃ«рЃЋ.)  
- рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃў рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃћрЃЉрЃў рЃЊрЃљ рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ

## рЃарЃљрЃбрЃЮрЃЏ рЃЕрЃЋрЃћрЃю
рЃЕрЃЋрЃћрЃю рЃњрЃЋрЃ»рЃћрЃарЃљ, рЃарЃЮрЃЏ рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃЉрЃўрЃќрЃюрЃћрЃАрЃў рЃўрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃА рЃўрЃАрЃћрЃЌрЃўрЃЋрЃћ рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА, рЃарЃЮрЃњрЃЮрЃарЃўрЃф рЃљрЃЦрЃЋрЃЌ рЃЏрЃАрЃЮрЃцрЃџрЃўрЃЮ рЃЎрЃЮрЃЏрЃърЃљрЃюрЃўрЃћрЃЉрЃА.  
**ERM-System** рЃњрЃљрЃФрЃџрЃћрЃЋрЃЌ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА, рЃЏрЃфрЃўрЃарЃћ рЃ«рЃљрЃарЃ»рЃћрЃЉрЃўрЃЌ рЃЏрЃўрЃўрЃдрЃЮрЃЌ рЃФрЃџрЃўрЃћрЃарЃў, рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃЊрЃљ рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ.

## рЃарЃЮрЃњрЃЮрЃа рЃЊрЃљрЃЋрЃўрЃгрЃДрЃЮ?
­ЪЉЅ рЃЊрЃљрЃўрЃгрЃДрЃћ рЃљрЃЦ: [рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃњрЃќрЃљрЃЏрЃЎрЃЋрЃџрЃћрЃЋрЃў](user-guide/intro.md)  
­ЪЉЅ рЃњрЃљрЃўрЃњрЃћ рЃЏрЃћрЃбрЃў рЃЕрЃЋрЃћрЃюрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ: [рЃЕрЃЋрЃћрЃюрЃў рЃЏрЃўрЃќрЃюрЃћрЃЉрЃў](about/goals.md)
